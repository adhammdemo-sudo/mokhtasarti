# Mokhtasarti - Next.js + Tailwind Starter (v2)

Enhanced starter with components, /admin and /game pages.

## Quick start (local)

1. Install dependencies
   ```bash
   npm install
   ```

2. Run dev server
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000
